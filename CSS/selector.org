#+SUBTITLE: CSS selectors are patterns used to select the element(s) you want to style in your web page
#+KEYWORDS: Selector
#+CATEGORIES: CSS

* Universal Selector
'*' will select all elements. Always we use this to reset some browser settings.
#+begin_src css
  ,* {
      margin: 0;
      padding: 0;
  }
#+end_src

* Attribute Selector
#+begin_src css
  [type="text"] {}
#+end_src

* Pseudo-class Selector
#+begin_src css
  a:hover {}
#+end_src

* Pseudo-element Selector
#+begin_src css
  p::first-line {}
#+end_src

* Descendant Selector
Select all p inside the div. Not Only its children.
#+begin_src css
  div p {}
#+end_src

* Children Selector
Select ul's all direct children li
#+begin_src css
  ul > li {}
#+end_src

* Adjacent Sibling Selector
Select first p after h1
#+begin_src css
  h1 + p {}
#+end_src

* General Sibling Selector
Select all p after h1
#+begin_src css
  h1 ~ p {}
#+end_src
